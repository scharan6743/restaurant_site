# Restaurant Website

A simple restaurant website built using Flask.

## Features
- Home Page
- Food Menu
- Add to Cart
- Cart Page
- Responsive Design

## Technologies Used
- Python
- Flask
- HTML
- CSS

## Run the Project

1. Install Flask
pip install flask

2. Run
python app.py

3. Open
http://127.0.0.1:5000
